# 🧮 Calculator Project

A simple and responsive Calculator built using HTML, CSS, and JavaScript. This calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear Screen (C)
- Responsive Design
- User-Friendly Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
